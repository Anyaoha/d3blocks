Chord

Chord block.

Description

A chord represents flows or connections between several entities or nodes. Each entity is represented by a fragment on the outer part of the circular layout. Then, arcs are drawn between each entity. The size of the arc is proportional to the importance of the flow.

param df

Input data containing the following columns: ‘source’ ‘target’ ‘weight’

type df

pd.DataFrame()

param title

Title of the figure.

type title

String, (default: None)

param filepath

File path to save the output

type filepath

String, (Default: user temp directory)

param figsize

Size of the figure in the browser, [width, height].

type figsize

tuple, (default: (800, 600))

param showfig

Open the window to show the chart.

type showfig

bool, (default: True)

param overwrite

Overwrite the output html in the destination directory.

type overwrite

bool, (default: True)

returns

df – DataFrame.

rtype

pd.DataFrame()

Examples

>>> # Load d3blocks
>>> from d3blocks import D3Blocks
>>> #
>>> # Initialize
>>> d3 = D3Blocks()
>>> #
>>> # Load example data
>>> df = d3.import_example('stormofswords')
>>> #
>>> # Plot
>>> d3.chord(df)
