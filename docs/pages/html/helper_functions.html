The D3Blocks library also contains various helper functions to get the data in the right shape for the desired chart.

vec2adjmat

Convert source and target into adjacency matrix.

param source

The source node.

type source

list

param target

The target node.

type target

list

param weight

The Weights between the source-target values

type weight

list of int

param symmetric

Make the adjacency matrix symmetric with the same number of rows as columns. The default is True.

type symmetric

bool, optional

param aggfunc

Aggregate function in case multiple values exists for the same relationship. ‘sum’ (default)

type aggfunc

str, optional

returns

adjacency matrix.

rtype

pd.DataFrame

Examples

>>> # Initialize
>>> d3 = D3Blocks()
>>> #
>>> # Load example
>>> df = d3.import_example('energy')
>>> #
>>> # Convert to adjmat
>>> adjmat = d3.vec2adjmat(df['source'], df['target'], df['weight'])

adjmat2vec

Convert adjacency matrix into vector with source and target.

param adjmat

Adjacency matrix.

type adjmat

pd.DataFrame()

param min_weight

edges are returned with a minimum weight.

type min_weight

float

returns

nodes that are connected based on source and target

rtype

pd.DataFrame()

Examples

>>> # Initialize
>>> d3 = D3Blocks()
>>> #
>>> # Load example
>>> df = d3.import_example('energy')
>>> Convert into adjmat
>>> adjmat = d3.vec2adjmat(df['source'], df['target'], df['weight'])
>>> #
>>> # Convert back to vector
>>> vector = d3.adjmat2vec(adjmat)
