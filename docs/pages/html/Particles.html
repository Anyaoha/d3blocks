Particles

Particles graph.

Description

The particles plot is to turn any word into an interactive graph. With a mouse-move or touch, the particle bounce and then return to their original place. Various properties can be changed such as the bouncing, particle size, and colors. The original javascript is forked from Ian Johnson’s Block.

param text

String to be visualized

type text

string

param radius

Size of the particles.

type radius

float (Default: 3)

param collision

Response of the interaction. Higher means that more collisions are prevented.

type collision

float, (default: 0.1)

param fontsize

Text fontsize. When increasing: also increase width and slighly the spacing.

type fontsize

int (Default: 250)

param spacing

The number of particles that fit in the text. A larger spacing reults in less particles. A smaller spacing reults in more particles.

type spacing

int (Default: 10)

param cmap

Color schemes can be found here: https://observablehq.com/@d3/color-schemes ‘Turbo’, ‘Rainbow’, ‘Blues’, ‘Reds’, ‘Inferno’, ‘Magma’

type cmap

String (default: ‘Set2’)

param background

Background color.

type background

String (default: ‘#000000’)

param title

Title of the figure.

type title

String, (default: None)

param filepath

File path to save the output

type filepath

String, (Default: user temp directory)

param figsize

Size of the figure in the browser, [width, height].

type figsize

tuple, (default: (800, 600))

param showfig

Open the window to show the graph.

type showfig

bool, (default: True)

param overwrite

Overwrite the output html in the destination directory.

type overwrite

bool, (default: True)

rtype

None

Examples

>>> # Load d3blocks
>>> from d3blocks import D3Blocks
>>> #
>>> # Initialize
>>> d3 = D3Blocks()
>>> #
>>> # Plot
>>> d3.particles('D3blocks')
>>> #
>>> # Specify
>>> d3.particles('D3Blocks', filepath='D3Blocks.html', collision=0.05, spacing=7, figsize=[750, 150], fontsize=130, cmap='Turbo', background='#ffffff')
>>> #

Input Data

The input is a text string, such as “D3Blocks”
