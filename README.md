#### Installation (Pypi)
```bash
pip install d3blocks     # Normal installation
pip install -U d3blocks  # Force update
```

#### Installation (clone)
```bash
git clone https://github.com/d3blocks/d3blocks.git
cd d3blocks
pip install -U .
```  

#### Import d3blocks package
```python
from d3blocks import d3blocks
```
